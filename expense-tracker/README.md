## 🎯 Project Goal

Build a simple web app where students can **add, manage, and track daily expenses** while learning React fundamentals.

---

## 🚀 Features

### ➕ Add Expense

Users can add expenses with:

* Expense Name
* Amount
* Category
* Date

The expense instantly appears in the list.

---

### 📋 View Expense List

Each expense shows:

* Name
* Amount
* Category
* Date
* Delete button

---

### ❌ Delete Expense

Users can remove any expense from the list easily.

---

### 🧮 Total Expense Calculator

Automatically calculates total spending.

Example:

```
Total Expense: ₹3500
```

---

### 🔍 Filter by Category

Filter expenses by:

* All
* Food
* Travel
* Education

---

### 💾 Local Storage Support

Expenses are saved in **browser local storage**.

✔ Data stays even after refresh
✔ No backend required

---

### 🌙 Dark / Light Mode

Toggle between:

* Light Mode ☀️
* Dark Mode 🌙

---

## 🛠️ Tech Stack

* React JS
* Tailwind CSS
* JavaScript (ES6)
* HTML5
* CSS3
* Local Storage API

---

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/AniketBais/React-Project.git
cd expense-tracker
npm install
npm run dev
```
